Auto Bucket (BETA)

Automatically uses water bucket when you fall.
Don't rely solely on this module, since you can die just because of bad ping.
Module uses reach distance to check if it can use water bucket. If you use Reach module ensure you didn't setup too big distance.
Don't fall on waterloggable blocks like leaves, since you water cannot be placed on top of such blocks.
Module doesn't check if you already are falling into water.
There was plans to add other items like cobweb or slime block.
Speed Threshold
Module tries to use water bucket only if your speed (blocks/sec) if greater than specified value.
