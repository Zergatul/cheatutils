Zoom

Allows you to zoom to see distant objects.
Use Events Scripting module to assign key bindings. Example script: 
events.onHandleKeys(() => {
    if (input.isKeyDown("C"))
        zoom.start(10, 0.2);
    else
        zoom.stop();
});
Here:
10 - final FOV
0.2 - how quickly zoom (in seconds, can be zero)
